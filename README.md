# 📚 مكتبة (Maktaba)

**مكتبة** هو تطبيق سطح مكتب مفتوح المصدر لقراءة والبحث في الكتب العربية والقرآن الكريم، مبني بلغة **Python** وواجهة **GTK4** وموجّه لمستخدمي Linux.
الهدف من المشروع هو تقديم تجربة قراءة خفيفة وسريعة وواضحة، مع دعم عربي كامل (RTL) وبحث فعّال.

---

## ✨ المميزات الرئيسية

- **خفيف وسريع:** واجهة GTK4 مع استجابة عالية واستهلاك منخفض للموارد.
- **تنظيم بسيط للكتب:** يعتمد على بنية ملفات واضحة (`HTML/XHTML + meta.json`) بدون قاعدة بيانات معقدة.
- **دعم عربي كامل:** عرض من اليمين إلى اليسار مع تحسينات للخطوط والنصوص.
- **بحث متقدم عبر Recoll:** فهرسة للمكتبة مع نتائج سريعة.
- **تصميم وحداتي:** تقسيم واضح (التطبيق، الإعدادات، الخدمات، الواجهات) يسهل التوسعة والصيانة.

---

## 🏗️ تبويبات التطبيق

### 1) تبويب القرآن الكريم 🕌
- تصفح سور القرآن من قائمة جانبية.
- نسخ الآيات بعزو تلقائي (اسم السورة + رقم الآية).
- بحث خاص بالقرآن مع تمييز النتائج.

### 2) تبويب القراءة 📖
- عرض الكتب بصيغة HTML (بما فيها الكتب المقسمة لأجزاء).
- تنقل عبر شجرة العناوين (مثل H1/H2/H3).
- قراءة معلومات الكتاب من `meta.json`.

### 3) تبويب البحث 🔍
- البحث في كامل المكتبة أو ضمن قسم/كتاب محدد.
- عرض النتائج مع مقتطفات نصية.
- فتح النتيجة مباشرة داخل القارئ مع تمييز كلمات البحث.

### 4) تبويب البحث الدلالي 🧠
- مساحة مخصصة للبحث الذكي/الدلالي (قابلة للتطوير حسب الإمكانات المتاحة).

---

## 📂 مثال عملي لإضافة كتاب

أنشئ كتاباً داخل مجلد `books` بهذا الشكل:

```text
books/
└── حديث/
    └── رياض_الصالحين/
        ├── meta.json
        ├── part1.html
        └── part2.html
```

### مثال `meta.json`

```json
{
  "title": "رياض الصالحين",
  "author": "الإمام النووي",
  "death_year": "676هـ",
  "category": "حديث"
}
```

> ملاحظة: أسماء الملفات التي تحتوي على `_` تُعرض في الواجهة بشكل أنظف (مع مسافات).

## 🛠️ المتطلبات

- Python 3.10+
- GTK4
- PyGObject
- Recoll (للبحث والفهرسة)

---

## ▶️ التشغيل

من داخل مجلد المشروع:

```bash
python3 main.py
```

إذا كانت لديك بيئة افتراضية (`venv`) داخل المشروع، سيحاول `main.py` تفعيلها تلقائياً.

---

## 🔎 آلية الفهرسة والبحث

- يعتمد المشروع على Recoll للفهرسة.
- تتم مراقبة تغييرات المكتبة وتشغيل إعادة الفهرسة عند الحاجة.
- نتائج البحث تُفتح مباشرة في موضعها داخل القارئ.

---

## 💡 ملاحظات للمطورين

- الإعدادات تحفظ في:
  - `~/.config/recoll-gtk/config.json`
- قاعدة فهرس Recoll المحلية تحفظ داخل المشروع في:
  - `.recoll/`
- فصل الواجهات داخل `views/` والخدمات داخل `services/` لتسهيل التوسعة.
