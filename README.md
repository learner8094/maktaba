# 📚 مكتبة (Maktaba)

**مكتبة** هو تطبيق سطح مكتب مفتوح المصدر لقراءة الكتب العربية والبحث فيها، مع تبويب خاص للقرآن الكريم، مبني باستخدام **Python + GTK4** لمستخدمي Linux.

---

## ✨ المزايا الحالية

- واجهة GTK4 خفيفة مع دعم اتجاه النص العربي (RTL).
- قارئ كتب بصيغة `HTML/XHTML` مع شجرة عناوين للتنقل السريع.
- تبويب قرآن مدمج مع نسخ الآيات بعزو السورة ورقم الآية.
- بحث نصي عبر المكتبة باستخدام **Recoll**.
- بحث دلالي (Semantic Search) كتجهيز قابل للتطوير.
- بنية كتب بسيطة بدون قاعدة بيانات معقدة (`meta.json` + ملفات HTML).

---

## 🧱 المتطلبات

- Python `3.10+`
- GTK4 + PyGObject
- Recoll (للفهرسة والبحث)

> ملاحظة: طريقة تثبيت GTK4/PyGObject تختلف حسب التوزيعة. ثبّت الحزم النظامية المناسبة أولاً، ثم شغّل التطبيق.

---

## ▶️ التشغيل (للمطورين ومن المصدر)

من داخل مجلد المشروع:

```bash
./scripts/maktaba
```

أو مباشرة:

```bash
python3 main.py
```

وإذا ثبّت المشروع كحزمة بايثون (يوفّر `console_scripts`):

```bash
maktaba
```

---

## 📦 التثبيت للمستخدم النهائي

### Flatpak (الخيار الموصى به)

1. ثبّت Flatpak في توزيعتك.
2. أضف Flathub.
3. ابحث عن التطبيق باسم **Maktaba** أو بالمعرّف:

   ```
   io.github.maktaba
   ```

4. ثبّت التطبيق وشغّله من قائمة التطبيقات.

تفاصيل التغليف موجودة في مجلد `packaging/`.

## ➕ إضافة كتاب جديد

أنشئ كتاباً داخل `books/` بالشكل التالي:

```text
books/
└── حديث/
    └── رياض_الصالحين/
        ├── meta.json
        ├── part1.html
        └── part2.html
```

مثال `meta.json`:

```json
{
  "title": "رياض الصالحين",
  "author": "الإمام النووي",
  "death_year": "676هـ",
  "category": "حديث"
}
```

## 🔎 الفهرسة والبحث

- يعتمد البحث على Recoll.
- يتم تحديث الفهرس محلياً حسب تغييرات المكتبة.
- يمكن فتح نتيجة البحث مباشرة داخل القارئ.
